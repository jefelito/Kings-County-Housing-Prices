# Mod_2-Project

Kings County Housing Prices Bakeoff includes the folllowing.

Exploratory Data Analysis (EDA): These visualizations should help someone unfamiliar with the data understand the target variable and the features that help explain that target variable.

Feature Engineering: Explains the features engineered and thought process behind why they would explain the selling price of the house.

Feature Selection: Showcase different techniques and combinations of feature selection.

Model Interpretation: Interpretation of coefficients.

There are multiple drafts of EDA and a CSV of the predictions useing the finalized model.
